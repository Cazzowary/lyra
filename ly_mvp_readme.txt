Ly_mvp.readme

Overview:
    Lyra is a tool that allows a user to create, search, and share an online astronomy journal.

User stories:
    Users create a profile with their name, equipment, and observing locations.
    Users will be able to log into the application.
    Users will be able to fill in a standardized observation form.
    Users will be able to select from their profile's equipment and locations.
        *Equipment checkbox, location drop-down.
    Users will be able to upload pictures, sketches, and scans to the form.
    Users will be able to save these forms, search them, and share them with other users.

Build:
    Frontend HTML/CSS/Bootstrap
    Backend Javascript/python
    Datbase tba
